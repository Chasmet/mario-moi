# Super Moi Bros

Mini jeu platformer style Super Mario avec **ta tête** à la place de Mario !

## Comment jouer

1. Clone le repo :
   ```bash
   git clone https://github.com/Chasmet/mario-moi.git
   cd mario-moi
   ```

2. Crée un dossier `assets` dans le projet

3. Sauvegarde ta photo (selfie) dedans et renomme-la exactement **`player.png`**

4. Installe pygame :
   ```bash
   pip install pygame
   ```

5. Lance le jeu :
   ```bash
   python main.py
   ```

Contrôles :
- Flèches ou Q / D : bouger
- Espace ou Haut : sauter
